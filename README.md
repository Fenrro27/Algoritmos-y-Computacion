# ModelosDeComputacion
# 📘 UHU - Ingeniería Informática

Repositorio con el contenido de las prácticas y el material teórico de las asignaturas del Grado en Ingeniería Informática de la **Universidad de Huelva (UHU)** en las que se ven temas de computación.  

Además de las soluciones prácticas, se incluye un proyecto unificado de **[Libro de Temarios Teóricos](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Temario/README.md)** desarrollado en LaTeX para consolidar la base conceptual de las distintas asignaturas.

## 🛠️ Tecnologías y Lenguajes de Programación

En este repositorio se hace uso de una amplia variedad de lenguajes de programación y herramientas según el paradigma y naturaleza de cada asignatura:

*   **C++**: Lenguaje principal empleado en **[Fundamentos de Análisis de Algoritmos](Fundamentos-de-Análisis-de-Algoritmos/README.md)** para el análisis empírico de búsquedas, potencias y ordenaciones.
*   **Java**: Utilizado en **[Sistemas Inteligentes](Sistemas-Inteligentes/README.md)** (resolutores CSP, agentes del framework GVGAI y planificadores) y en **[Aprendizaje Automático](Aprendizaje-Automatico/README.md)** (controladores y recolectores para el simulador TORCS).
*   **Python (y Jupyter Notebooks)**: Utilizado para modelado en **[Aprendizaje Automático](Aprendizaje-Automatico/README.md)**, implementación de metaheurísticas (QAP) en **[Modelos Bioinspirados](Modelos-Bioinspirados-y-Heuristicas-de-Busqueda/README.md)**, y scripts auxiliares de trazado gráfico.
*   **Haskell**: Empleado en **[Modelos Avanzados de Computación](Modelos-Avanzados-de-Computacion/README.md)** para la resolución funcional de algoritmos y árboles binarios.
*   **Prolog**: Utilizado en **[Representación del Conocimiento](Representación-del-Conocimiento/README.md)** para la programación lógica de grafos, espacios de estados, árboles y programación por restricciones CLP(FD).
*   **CLIPS**: Empleado en **[Inteligencia Artificial](Inteligencia-Artificial/README.md)** para la creación de sistemas expertos de diagnóstico basados en reglas de producción.
*   **LaTeX**: Empleado para la redacción y compilación técnica del **[Libro de Temarios Teóricos](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Temario/README.md)** consolidado.
