## 🛠️ Tecnologías y Lenguajes de Programación

En este repositorio se hace uso de una amplia variedad de lenguajes de programación y herramientas según el paradigma y naturaleza de cada asignatura:

*   **C++**: Lenguaje principal empleado en **[Fundamentos de Análisis de Algoritmos](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Fundamentos%20de%20An%C3%A1lisis%20de%20Algoritmos/README.md)** para el análisis empírico de búsquedas, potencias y ordenaciones.
*   **Java**: Utilizado en **[Sistemas Inteligentes](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Sistemas%20Inteligentes/README.md)** (resolutores CSP, agentes del framework GVGAI y planificadores) y en **[Aprendizaje Automático](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Aprendizaje%20Automatico/README.md)** (controladores y recolectores para el simulador TORCS).
*   **Python (y Jupyter Notebooks)**: Utilizado para modelado en **[Aprendizaje Automático](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Aprendizaje%20Automatico/README.md)**, implementación de metaheurísticas (QAP) en **[Modelos Bioinspirados](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Modelos%20bioinspirados%20y%20heuristicas%20de%20busqueda/README.md)**, y scripts auxiliares de trazado gráfico.
*   **Haskell**: Empleado en **[Modelos Avanzados de Computación](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Modelos%20Avanzados%20de%20Computacion/README.md)** para la resolución funcional de algoritmos y árboles binarios.
*   **Prolog**: Utilizado en **[Representación del Conocimiento](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Representaci%C3%B3n%20del%20Conocimiento/README.md)** para la programación lógica de grafos, espacios de estados, árboles y programación por restricciones CLP(FD).
*   **CLIPS**: Empleado en **[Inteligencia Artificial](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Inteligencia%20Artificial/README.md)** para la creación de sistemas expertos de diagnóstico basados en reglas de producción.
*   **LaTeX**: Empleado para la redacción y compilación técnica del **[Libro de Temarios Teóricos](file:///c:/Users/kjba2/Documents/Algoritmos-y-Computacion/Temario/README.md)** consolidado.
